# PeachyKareoke

PeachyKareoke 是一款基于 Electron 和 React 构建的现代跨平台卡拉OK应用程序。它为您带来录音室级别的音轨分离、实时变调以及高级音频路由功能。

## 主要功能

- **自动化音轨分离**：只需将任何 MP3 或 WAV 文件拖放至应用内，即可自动将歌曲分离为高质量的伴奏和人声音轨。
- **高级音频矩阵**：可将麦克风、伴奏音轨和人声音轨独立路由到不同的物理输出设备（例如：观众扬声器与您的监听耳机）。
- **零延迟实时变调**：使用行业标准的 SoundTouch 算法，通过优化的 AudioWorklet 实时调整伴奏或人声的音高，而不会改变歌曲速度。
- **监听延迟补偿**：使用实时监听偏移滑块，完美对齐您的实时麦克风输入与伴奏音轨。
- **歌词与注释**：内置 LRC 歌词查看器与内联编辑器，助您完美呈现每一次演唱。

## 安装与使用

1. 从 Releases 页面下载最新版本的安装包。
2. 运行安装程序并启动 PeachyKareoke。
3. 使用 **设备设置** 面板选择您的麦克风、观众扬声器和监听耳机。
4. 将歌曲拖放至左侧的歌曲列表中以开始处理。
5. 点击播放！
