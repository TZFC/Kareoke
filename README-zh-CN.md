# 🍑 桃子卡拉OK (PeachyKareoke)

欢迎使用 **桃子卡拉OK** (PeachyKareoke)，这是一款为歌手、演员和主持人设计的简单且对非技术用户友好的卡拉OK播放应用程序。

该应用程序可以将任意歌曲分离为**人声**和**伴奏**，并允许您将它们完美同步地输出到不同的音频设备。

---

## 🌟 核心特色

* **双路音频输出：** 将分离出的人声音轨发送到您的监听耳机中（以便您清晰地听到自己的歌声），同时仅将伴奏音轨和您的麦克风声音（带有人声混响/均衡器特效）播放到观众音响！
* **专业级人声特效：** 实时调节麦克风的音量、低音（Bass）、高音（Treble），并添加高品质混响（回声）。
* **智能变调：** 自由升高或降低歌曲的主调，且完全不会改变音乐的播放速度。
* **歌词同步滚动：** 支持加载标准的 `.lrc` 时间戳歌词文件，播放歌曲时歌词会自动高亮并滚动。
* **零配置极简上手：** 无需复杂的第三方软件、虚拟声卡或修改系统音频驱动，双击打开应用，插上设备即可开唱！

---

## 🚀 用户快速上手指南

1. **启动应用程序：**
   * 双击释放文件夹中的 `PeachyKareoke.exe` 启动程序。
2. **添加歌曲：**
   * 直接将任何 `.mp3` 或 `.wav` 音频文件拖拽到屏幕上的拖放区域中。
3. **选择并配置您的音频设备：**
   * **麦克风输入 (Microphone Input)：** 选择您唱歌使用的麦克风设备。
   * **观众音响 (Audience Speaker)：** 选择房间主音响或外放喇叭（播放伴奏 + 麦克风人声）。
   * **歌手耳返 (Singer Monitor)：** 选择您佩戴的监听耳机（播放歌手人声 + 伴奏）。
4. **自定义混音设置：**
   * 拖动滑块调整伴奏和人声的独立音量。
   * 使用 **升高 1 半音** / **降低 1 半音** 按钮改变歌曲的调式。
   * 在右下角的 **麦克风实时混音** 面板中，微调麦克风音量、高低音及混响深度。
5. **添加歌词：**
   * 切换到 **歌词 (Lyrics)** 标签页，点击 **编辑歌词 (Edit Lyrics)**，将您的 `.lrc` 格式歌词粘贴进去，点击 **编辑完成 (Done Editing)**。歌词会随播放位置自动滚动！
