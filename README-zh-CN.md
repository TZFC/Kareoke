# Peachy Karaoke (桃子K歌)

欢迎使用 **Peachy Karaoke**，这是一款完全离线、基于人工智能的终极 K 歌应用。只需将您最喜欢的歌曲拖拽到应用中，Peachy 就会在您的设备上自动分离人声、提取歌词并生成简谱——一切都在本地完成！

![Peachy Karaoke](media/screenshot.png)

## 功能特色

### 🎙️ AI 音频分离
Peachy Karaoke 内置了业界领先的 **Demucs** AI 模型。
当您将 MP3 或 WAV 文件拖入应用时，它会自动将伴奏与人声分离，为您提供完美的 K 歌伴奏。

### 📜 AI 歌词生成
没有歌词？没问题。Peachy 利用 OpenAI 强大的 **Whisper** 语音识别 AI（针对本地运行进行了优化），自动从提取的人声中生成带有精准时间轴的 `.lrc` 歌词。

### 🎵 AI 简谱生成
无论您是在练习音准还是学习新歌，Peachy 都能使用 **Spotify Basic Pitch** 追踪人声的音高，并自动将其转换为**简谱 (Jianpu)**。

### 🎛️ 实时音频处理与路由
- **硬件路由矩阵：** 将您的麦克风独立路由到耳机（监听）或主扬声器（观众）。
- **实时混响与 EQ：** 实时为您的人声添加录音室级别的混响、低音和高音调节。
- **音频限幅器：** 内置动态压缩器（Dynamics Compressor），在您大声演唱时防止音频爆音和失真。
- **音高与音量控制：** 独立调整伴奏和人声的音调和音量，以匹配您的音域。

### 🎨 动态主题
- **系统集成：** 根据您的 Windows 系统偏好设置自动在深色和浅色模式之间切换。

### 💾 自动保存配置
一旦您为某首歌曲调好了完美的音调、混响或歌词偏移，Peachy 就会记住它！所有配置都会按歌曲自动保存。

## 使用指南

1. **启动应用：** 打开 Peachy Karaoke。
2. **添加音乐：** 将任何 `.mp3` 或 `.wav` 文件直接拖拽到应用窗口中。
3. **等待 AI 处理：** 应用会显示进度条，在后台分离音轨、转录歌词和计算简谱。
4. **开始演唱：** 从下拉菜单中选择您的麦克风，调整混响，尽情享受吧！

## 系统要求
- Windows 10 / 11
- 至少 4GB 内存（建议 8GB+ 以加快 AI 处理速度）
- 无需连接互联网！所有 AI 处理均 100% 离线进行。
